Папка для пользовательских миров Gazebo.

Как использовать:

1. Скопируйте свой `.world` файл в эту директорию (при желании добавьте модели рядом или пропишите абсолютные пути в world).
2. Проверьте, что камера в мире видит робота. Если камера уже публикует TF/топики, отключите наши статические TF: `publish_static_camera_tf:=false`.
3. Запускайте симуляцию и трекер с указанием своего мира:
   ```bash
   roslaunch sim_pkg demo.launch world_file:=$(find sim_pkg)/worlds/custom/your_world.world publish_static_camera_tf:=false
   ```
4. Если робот уже прописан в вашем мире, выключите автоспавн: `spawn_robot:=false`. Если нужен наш робот, оставьте значение по умолчанию или укажите свой SDF через `robot_model:=...`.
